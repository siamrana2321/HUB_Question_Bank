# HUB Question Bank – User Manual

Welcome to the **HUB Question Bank**.

The question bank is now hosted on GitHub to provide a centralized, organized, and easily accessible collection of previous examination question papers for all students of Hamdard University Bangladesh.

---

# Repository Structure

All question papers are organized **semester-wise** for easy navigation.

```
HUB_Question_Bank
│
├── Semester 1
├── Semester 2
├── Semester 3
├── Semester 4
├── Semester 5
├── Semester 6
├── Semester 7
└── Semester 8
```

Each semester folder contains the available PDF question papers for the corresponding courses.

---

# How to View a Question Paper

1. Open the GitHub repository.
2. Select your semester folder.
3. Open the desired course PDF.
4. GitHub will display the PDF directly in your browser.

---

# How to Download a Question Paper

### Method 1 – Download a Single PDF

1. Open the required semester folder.
2. Click on the desired PDF file.
3. Select the **Download** button (or **Raw** → Save As, depending on your browser).
4. Save the file to your device.

---

### Method 2 – Download the Entire Question Bank

1. Open the repository homepage.
2. Click the green **Code** button.
3. Select **Download ZIP**.
4. Extract the ZIP file on your computer.

This will download all available question papers at once.

---

# Important Notes

* The repository is updated periodically as new question papers become available.
* If a particular course or semester is missing, it means the question paper has not yet been added.
* Always check the repository for the latest updates before examinations.
* Students are encouraged to report missing or incorrect files to their Class Representative (CR).

---

# Support

For any issues, missing question papers, or suggestions, please contact your respective **Class Representative (CR)**.

The repository is maintained to ensure that all students have quick and organized access to previous examination question papers.

Happy Studying!
